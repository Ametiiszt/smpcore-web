SMPCore

The best SMP plugin

A plugin designed to make creating and managing Minecraft SMP servers easier


Explosion Control

Enable or disable specific explosion types per world. Disabled explosions are cancelled and the explosive item will be dropped.

Enchant Control

Define maximum enchantment levels or completely disable specific enchants. Disabled enchantments cannot be obtained.

Item Control

Set the maximum item amounts per player or completely disable them. Disabled items cannot be obtained.

Potion Control

Set the maximum potion type amounts per player or completely disable them. Disabled potion types cannot be obtained.

Crafting Recipes

Enable custom crafting recipes such as Easy Gap, Enchanted Golden Apple, or Easy Anvil. Each recipe can be individually toggled in the config.yml.

Dimension Control

The end and nether dimensions can be disabled.

One Mace

If enabled, only one mace can be crafted on the server.
